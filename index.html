Commandos GIT

  • git init - Inicializa um novo repositorio Git
  • git add nomearquivo|. - adiciona item no stage
  • git status - exibe o estado dos arquivos da workingtree
  • git log - exibe log de commits da branch(histria atual)
  • git log --online - exibe log de commits de forma resumida
  • git config user.name "nome" - define o nome do usuário
  • git config user.email "email" - define o nome do email
  • git commit -m "mensagem do commit "
  • git diff - exibe a diferença entre wk e HEAD
  • git checkout -b novabranch [branch_base]
  • git checkout branch_alvo - muda para branch_alvo
  • git branch - exibe as branchs locais
  • git merge branch_a_ser_adicionada
